How to use this map

Measuring

Hold down [m] to create a path. While holding down the [m] key, when you click on the map the point where you click will be added to the current path. When you release the [m] key the path is completed. Pressing the [m] key again will start a new path.

The measurements modal (which you can toggle by clicking on Measurements in the top right corner) displays all of the paths you’ve made. The table displays each point you clicked, along with its distance from the previous point, and a final tally of the total length of the path (all measurements are in kilometers).

Views

Use the dropdown box in the bottom left corner to choose which view you’d like to see.

Measurements

You haven’t taken any measurements yet. Hold [m] and click on the map to define a path. The distance of that path will be recorded here. The path will be complete when you release the [m] key.
